# watch list

自己动手开发一个简单的 Watchlist 程序

## setup

create a virutal env

```bash
python -m venv venv
```

active the virutal env

```bash
source ./.venv/bin/activate
```

> fish is not support

install dependencies

```bash
pip install -r requirements.txt
```

run

```bash
python app.py
```
